# The three views

Every block has three modes in the top bar:

- **Read** — the clean finished document; embeds dissolved inline so a composed
  block reads as one seamless page.
- **Blocks** — the working view; each embed is a card you can click to edit in
  place, and you can add child blocks.
- **Edit** — the raw Markdown (title + body), with the `[[` picker and the
  **Carve selection** button.

Use the sidebar's **Roots / All** toggle to list top-level blocks or every block.
