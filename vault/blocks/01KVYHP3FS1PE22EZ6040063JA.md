# Installing the prerequisites

![[01KVYHAT6GF9C0NB8AHPAAKX6Y]]

Each block below installs everything, in order, for one platform.

## Debian / Ubuntu

```sh
sudo apt update
sudo apt install -y build-essential curl wget file libssl-dev libxdo-dev \
  libwebkit2gtk-4.1-dev libayatana-appindicator3-dev librsvg2-dev
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y   # Rust
. "$HOME/.cargo/env"
cargo install just tauri-cli
```

## Fedora

```sh
sudo dnf group install -y "C Development Tools and Libraries"
sudo dnf install -y curl wget file openssl-devel libxdo-devel \
  webkit2gtk4.1-devel libappindicator-gtk3-devel librsvg2-devel
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y   # Rust
. "$HOME/.cargo/env"
cargo install just tauri-cli
```

## Arch / Manjaro

```sh
sudo pacman -Syu --needed base-devel curl wget file openssl xdotool \
  webkit2gtk-4.1 libappindicator-gtk3 librsvg
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y   # Rust
. "$HOME/.cargo/env"
cargo install just tauri-cli
```

## openSUSE

```sh
sudo zypper install -t pattern devel_basis
sudo zypper install -y curl wget file libopenssl-devel \
  webkit2gtk3-devel libappindicator3-devel librsvg-devel
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y   # Rust
. "$HOME/.cargo/env"
cargo install just tauri-cli
```

## macOS

The WebView is the system WKWebView — nothing to install for it.

```sh
xcode-select --install                                                   # Clang + macOS SDK
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y   # Rust
. "$HOME/.cargo/env"
cargo install just tauri-cli
```

## Windows

PowerShell; WebView2 ships with Windows 11 and current Windows 10.

```powershell
winget install --id Microsoft.VisualStudio.2022.BuildTools -e `
  --override "--add Microsoft.VisualStudio.Workload.VCTools --includeRecommended"
winget install --id Rustlang.Rustup -e                                   # Rust
winget install --id Microsoft.EdgeWebView2Runtime -e                     # if not already present
cargo install just tauri-cli
```
