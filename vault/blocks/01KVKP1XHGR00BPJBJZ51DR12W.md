### Find the source block — don't edit the file

A generated file opens with a banner naming the block it came from:

`<!-- @generated by mkb from vault block <ULID> (<Title>). … -->`

That `<ULID>` is your edit target. To change anything in the file, edit **that block** — never the
file. If the text you want to change lives in an *embedded* section rather than the page block
itself, follow the trail: read the page block, find which child block holds that text, and edit
the child. The change then flows into **every** doc that embeds it.
